# Breach Point

A third-person shooter built in Unreal Engine 5.

## Features
- Level-based progression (5 levels)
- Weapon unlocks per level
- Enemy patrol and shooting AI
- Interactive environment (triggers, doors, movable objects)
- Graphics, sound, and control settings

## Controls
- WASD – Move
- Mouse – Aim / Shoot
- R – Reload
- Esc – Pause

## Engine
Unreal Engine 5.7.1
